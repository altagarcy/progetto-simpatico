# UniStay

Sistema di prenotazione di alloggi sviluppato per il corso di Programmazione Object Oriented e Basi di Dati.
Il progetto si trova nella fase iniziale di analisi e progettazione del dominio. Sono stati definiti il class diagram e le principali entità del sistema, tra cui utenti (Host e Guest), annunci, prenotazioni, pagamenti e recensioni.
Le fasi successive prevedono la progettazione del database e l'implementazione dell'applicazione Java secondo il pattern BCE + DAO.
